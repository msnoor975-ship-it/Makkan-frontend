# Image Naming Guide

## Folder Structure

### `/public/images/backgrounds/`
- `hero-bg.jpg` - Main hero background (1920x1080px)
- `pattern-subtle.png` - Subtle pattern for section backgrounds
- `about-bg.jpg` - About section background
- `contact-bg.jpg` - Contact section background

### `/public/images/hero/`
- `hero-house-1.jpg` - Featured house for homepage hero (1920x600px)
- `hero-house-2.jpg` - Secondary hero image
- `hero-house-3.jpg` - Tertiary hero image
- `lifestyle-1.jpg` - People enjoying home life
- `lifestyle-2.jpg` - Family in modern home

### `/public/images/agents/`
- `default-agent.jpg` - Default agent photo for homepage (400x400px square)
- `agent-1.jpg` - Individual agent photos
- `agent-2.jpg` - Individual agent photos
- `agent-3.jpg` - Individual agent photos

### `/public/images/houses/`
- `default-house.jpg` - Default house image for listings without photos (1200x800px)
- `house-1.jpg` - Individual house photos
- `house-2.jpg` - Individual house photos
- `house-3.jpg` - Individual house photos

### `/public/images/icons/`
- `home.svg` - Home icon for navigation
- `house.svg` - House icon
- `user.svg` - User/customer icon
- `calendar.svg` - Calendar/reservation icon
- `settings.svg` - Settings icon
- `edit.svg` - Edit action icon
- `delete.svg` - Delete action icon
- `view.svg` - View action icon
- `add.svg` - Add/create icon
- `search.svg` - Search icon
- `check.svg` - Checkmark/available status
- `clock.svg` - Pending/reserved status
- `x.svg` - X mark/sold status

## Recommended Image Sources
- Unsplash.com (free high-quality photos)
- Pexels.com (free stock photos)
- Flaticon.com (free icons)

## Image Specifications
- Hero images: 1920x600px (landscape)
- House photos: 1200x800px (landscape)
- Agent photos: 400x400px (square)
- Profile photos: 400x400px (square)
- Icons: 24x24px or 32x32px (SVG preferred)
